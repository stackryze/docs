title: "Stackryze DNS"
description: "Modern DNS hosting for everyone — fast, free, and security-first."
---

Stackryze DNS is a free authoritative DNS hosting service, designed with security in mind. Running on open-source software, it's free for everyone to use.

<Note>
  The full service lives at [dns.stackryze.com](https://dns.stackryze.com). Status & live metrics are at [status.stackryze.com](https://status.stackryze.com).
</Note>
