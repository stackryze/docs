## Domains release history

Stackryze Domains went from closed beta in April 2026 to a stable 1.0 release in July 2026. Every release is documented below.

### 2026

| Version | Date | Highlights |
| --- | --- | --- |
| [v1.0.0](/changelog/domains/v1.0.0) | July 4, 2026 | First stable release — public registration, 7-day cooling-off, audit log, scoped API tokens, 5-domain quota. |
| [v0.7.0](/changelog/domains/v0.7.0) | June 10, 2026 | Open beta — captcha-free registration, multi-domain dashboard, 50 forwards per domain, renewal grace period. |
| [v0.6.0](/changelog/domains/v0.6.0) | May 20, 2026 | Wildcard DNS, custom nameservers, basic email forwarding, first API endpoints. |
| [v0.5.0](/changelog/domains/v0.5.0) | April 15, 2026 | Closed beta — `indevs.in` registration, default Stackryze nameservers, My Domains dashboard. |
