Azure DNS is Microsoft's DNS hosting service, with high availability and tight Azure integration.

## Quick setup

1. Create a Microsoft Azure account.
2. Create a DNS zone for your subdomain.
3. Note your Azure DNS nameservers.
4. Register with Stackryze Domains using your nameservers.

For detailed instructions, check back soon, join the Discord community, or contact support.
